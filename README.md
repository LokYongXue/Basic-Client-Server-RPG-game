# Basic-Client-Server-RPG-game

## ***Install required dependencies***  
In your Visual Studio Code terminal, run the following commands:  
```
pip install pygame
```

## ***Ip Address and Port***  
Modify the **SERVER_HOST** and ***SERVER_PORT*** in **config.py**

```
SERVER_HOST = '10.107.25.172'
SERVER_PORT = 54323
```
